THE CONVEYOR

Système de tri automatisé NFC et intégration WMS

Logistics Industrial

Vue d'ensemble du projet

THE CONVEYOR est un système de tri automatique de colis utilisant la technologie NFC et une intégration directe avec le système de gestion d'entrepôt Dolibarr. Le projet automatise complètement le routage des colis vers trois entrepôts différents (C, D, ou erreur B) basé sur leur référence NFC, optimisant ainsi la logistique industrielle.

THE CONVEYOR

Tri automatique par NFC

Système de convoyage automatisé avec tri NFC

Technologies utilisées

Arduino C/C++ NFC RFID Dolibarr WMS REST API Servo Motors IR Sensors

Intégration complète hardware/software pour l'automatisation industrielle, combinant Arduino pour le contrôle physique et l'API REST de Dolibarr pour la gestion des données.

Objectifs du projet

  • Automatisation complète : Éliminer la manipulation manuelle des colis avec un tri automatique 100% automatisé
  • Intégration WMS : Connexion directe avec Dolibarr pour routage intelligent basé sur les données d'inventaire
  • Tri multi-destination : Diriger automatiquement les colis vers 3 entrepôts (C, D, ou erreur B)
  • Haute précision : Atteindre plus de 98% de précision de tri grâce à la technologie NFC
  • Performance optimale : Traiter 100+ colis par heure avec temps de cycle minimisé

Processus de fonctionnement

1. Détection

Capteur infrarouge détecte l'arrivée d'un colis sur le convoyeur

2. Arrêt

Le convoyeur s'arrête automatiquement pour permettre la lecture

3. Lecture NFC

Identification du colis par lecture de l'étiquette NFC

4. Requête WMS

Consultation de la base de données Dolibarr via API REST

5. Tri

Servomoteur dirige le colis vers l'entrepôt approprié

6. Reprise

Le convoyeur reprend son cycle pour le colis suivant

Architecture technique

  • Contrôleur : Arduino programmé en C/C++ pour gestion du flux et des décisions
  • Capteurs : Module NFC RFID pour identification et capteurs infrarouges pour détection
  • Actionneurs : Servomoteurs pour diriger les colis + moteur DC pour le convoyeur
  • Communication : API REST pour interroger Dolibarr et obtenir les destinations
  • Interface : Monitoring en temps réel de l'état du système et des statistiques

Fonctionnalités clés
