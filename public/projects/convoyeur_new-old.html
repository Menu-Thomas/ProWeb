üè≠ THE CONVEYOR - Automated Sorting System

An automatic package sorting system using NFC technology and integration with the Dolibarr warehouse management system to optimize industrial logistics.

üì¶ System Description

THE CONVEYOR automates package sorting through an Arduino-based control system and NFC technology.

It integrates directly with the Dolibarr warehouse management system via its REST API to determine package routing.

Packages are automatically directed to one of three warehouses (C, D, or error B) according to their NFC reference.

Complete hardware/software integration for industrial automation.

üîÑ Operating Process

  1. Detection: Infrared sensor detects a package
  2. Stop: The conveyor pauses
  3. NFC Reading: Package identification by NFC tag
  4. WMS Query: Consultation of Dolibarr database
  5. Sorting: Servo motor directs to appropriate warehouse
  6. Resume: The conveyor resumes its cycle

üîß Technical Architecture

  • Controller: Arduino with C/C++ programming
  • Sensors: NFC RFID and infrared detectors
  • Actuators: Servo motors for sorting
  • Communication: REST API with Dolibarr
  • Interface: Real-time monitoring
