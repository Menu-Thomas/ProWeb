Trash Management

Système intelligent IoT + Réalité Augmentée

Smart City Project

Vue d'ensemble du projet

Trash Management est un système intelligent de gestion des déchets combinant technologies IoT et réalité augmentée pour optimiser la collecte et le tri des déchets urbains. Le projet utilise des capteurs de niveau de remplissage connectés et une application de visualisation AR pour améliorer l'efficacité des circuits de collecte et réduire les coûts opérationnels.

Trash Management

Gestion intelligente des déchets IoT + AR

Système de gestion des déchets avec capteurs IoT et visualisation AR

Technologies utilisées

Arduino Unity AR IoT Sensors C# WiFi/Bluetooth

Combinaison de capteurs IoT pour monitoring en temps réel et application Unity AR pour visualisation immersive des données de remplissage des poubelles.

Objectifs du projet

  • Monitoring en temps réel : Capteurs IoT mesurant niveau de remplissage des poubelles publiques
  • Visualisation AR : Application de réalité augmentée affichant données de remplissage au-dessus des poubelles
  • Optimisation circuits : Algorithme d'optimisation des tournées de collecte basé sur remplissage réel
  • Dashboard centralisé : Interface web de monitoring pour gestionnaires urbains
  • Réduction coûts : Diminuer collectes inutiles et optimiser consommation carburant

Dashboard de monitoring

  • Vue carte : Géolocalisation de toutes les poubelles avec code couleur selon remplissage
  • Statistiques temps réel : Nombre de poubelles pleines, taux de remplissage moyen, alertes actives
  • Optimisation tournées : Algorithme suggérant circuits de collecte optimaux
  • Rapports analytiques : Tendances de production de déchets par zone et période
  • Notifications : Alertes email/SMS quand intervention requise
