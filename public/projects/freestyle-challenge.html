Freestyle Challenge

Jeu web audio interactif mobile-first pour sessions de freestyle

Seazon Studios
Démo Live Code Source

Vue d'ensemble

Développement d'un jeu web audio interactif pour Seazon Studios, permettant aux utilisateurs de faire du freestyle sur des beats avec génération aléatoire de mots. L'application mobile-first offre une expérience ludique et facilement partageable, avec un CTA intégré pour réserver des sessions studio.

Gameplay et enregistrement

Expérience de jeu fluide avec compte à rebours, lecture seamless des beats et enregistrement micro isolé pour une qualité audio optimale.

Compte à rebours 3-2-1 avec signal sonore
Lecture seamless des beats en boucle (sans clic/pop)
Enregistrement micro isolé (beat routé vers haut-parleurs)
Affichage mots grand format avec preview du suivant
Système de scoring personnel "Used it!" interactif

Technologies audio avancées

Utilisation des dernières technologies web pour offrir une expérience audio professionnelle directement dans le navigateur.

Vanilla JavaScript Vite Web Audio API MediaRecorder API Mobile-First PWA Netlify Deploy
Compatible iOS Safari + Android Chrome (2 dernières versions)
Traitement audio local uniquement (pas de backend requis)
Single-page application optimisée pour performance
Analytics intégrés (session start/end, booking click)
