üì° Sensor Sensei - LoRa Extension for Sensor.Community

An extension of the Sensor.Community project using LoRa communication to enable deployment of environmental sensors in areas without direct WiFi.

üåç Project Concept

Sensor Sensei extends the Sensor.Community project by enabling LoRa communication between sensors and a central WiFi relay node.

This solution is ideal for deployments in remote or distributed environments where direct WiFi is not available for each sensor node.
